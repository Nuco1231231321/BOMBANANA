## 三、YouTube 视频字幕分析 — 用户真实需求

从官方视频 `Defuse while being Deaf, Blind, and Mute!` 提取的关键玩法要素:

### 3.1 玩家实际遇到的问题 (视频中展示的)

1. **不理解manual规则** → "I don't understand how to tell... You need to know the color of the light, apparently"
2. **手势系统混乱** → "What kind of gestures can you do?" / "We need to come up with a system"
3. **信息传递链断裂** → Deaf Monkey 描述不清 → Mute找不到规则 → Blind瞎猜
4. **时间压力导致panic** → "We've got 3 seconds left, so we're probably going to blow up"
5. **LED颜色是关键信息但容易被忽略** → 视频中两次提到需要LED颜色
6. **wire编号系统混乱** → "One, two, or three, Ash? Which one do I cut?"

### 3.2 视频中展示的游戏机制 (我们的内容缺失这些)

| 游戏机制 | 视频展示了什么 | 我们网站有没有 |
|---|---|---|
| Wire模块 3线版 | 蓝/绿/黄 + LED色决定剪哪根 | ❌ 没有具体规则 |
| Wire模块 4线版 | 绿/红/黄/蓝 + 蓝色LED → 剪第三个(黄) | ❌ 没有 |
| 手势系统 | emote wheel, 举手, 指方向 | ❌ 完全没有 |
| 呼叫流程 | "module name → wire colors → LED color → Mute查表 → gesture → cut" | ❌ 部分有但不清晰 |
| 团队角色轮换 | 第一次失败后换角色重试 | ❌ 没有 |
