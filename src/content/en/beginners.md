## Getting Started with BOMBANANA!

Welcome to the world's most chaotic bomb defusal squad. This guide will get you from zero to your first successful defusal.

### Step 1: Get the Game

1. Go to the [BOMBANANA! Demo on Steam](https://store.steampowered.com/app/4747510/BOMBANANA_Demo/)
2. Click **Download** (it's free!)
3. Install and launch

### Step 2: Find Two Friends

BOMBANANA! requires exactly **3 players**. No more, no less. Each player picks one role:

- 🙈 **Blind Monkey** — touches the bomb (hardest to mess up, most stressful)
- 🙉 **Deaf Monkey** — the communication bridge (requires calm under pressure)
- 🙊 **Mute Monkey** — reads the manual (requires clear thinking)

### Step 3: Set Up Communication

Before your first bomb, spend 5 minutes with your team:

1. Agree on **gesture meanings** (especially Mute → Deaf)
2. Practice **2-3 key gestures**
3. Establish **confirmation signals** (yes/no/repeat)

### Step 4: Start with Easy Bombs

The first few levels have simple modules. Use them to:
- Get comfortable with your role
- Build your team's communication rhythm
- Learn how the manual is organized

### Step 5: Debrief After Each Bomb

After each attempt (success or explosion):
- What went well?
- What was confusing?
- Which gestures need improvement?

## Role Selection for Beginners

| If You Are... | Try This Role |
|---------------|---------------|
| Good at giving clear instructions | Deaf Monkey |
| Good at puzzles and manuals | Mute Monkey |
| Good under pressure, steady hands | Blind Monkey |
| New to gaming | Blind Monkey (simplest mental load) |

## First Bomb Walkthrough

1. **Blind Monkey** feels the bomb layout and reports module count
2. **Deaf Monkey** visually identifies each module type
3. **Mute Monkey** looks up the first module in the manual
4. **Mute → Deaf → Blind** chain begins for Module 1
5. Repeat for each module
6. Celebrate or explode — either way, learn from it!

## Common Beginner Mistakes

- **Talking over each other** — only the Deaf Monkey should speak
- **Skipping confirmations** — always have Blind repeat instructions back
- **Rushing** — panic is your real enemy, not the timer
- **Not practicing gestures** — 5 minutes of prep saves 50 minutes of frustration
