## BOMBANANA! First Session Checklist

A printable checklist for your first session. Go through each item before hitting "Start" — 5 minutes of prep prevents 50 minutes of frustration.

![Pre-Game Setup](/images/screenshots/ss_68f3c44625be8227496c6c017bfe847ed3c83e3d.thumb.jpg)

## Technical Setup

### All Three Players

- [ ] Steam installed and logged in with a valid account
- [ ] BOMBANANA! Demo downloaded (Steam App ID: 4747510)
- [ ] Game launched at least once to confirm it runs
- [ ] Headphones plugged in (not speakers — prevents echo)
- [ ] Microphone working (test in Steam → Settings → Voice)
- [ ] Steam overlay enabled (Steam → Settings → In Game)
- [ ] Discord/other VC apps closed (use in-game voice for intended experience)
- [ ] Stable internet connection (no downloads running in background)

### Host Player Only

- [ ] NAT type is not "Strict" (check in Steam → Settings → In Game → Steam Networking)
- [ ] No firewall blocking Steam (Windows Defender should auto-allow)

## Role Assignment

- [ ] **Blind Monkey assigned** — This player will touch the bomb
- [ ] **Deaf Monkey assigned** — This player will see everything and give instructions
- [ ] **Mute Monkey assigned** — This player will read the manual and gesture
- [ ] Each player understands their role's [capabilities and limitations](/roles)

## Communication Prep

- [ ] **Color system agreed:** How will the Mute indicate colors? (Number mapping: 1=red, 2=blue, 3=yellow / Color cards / Pointing to objects)
- [ ] **Number gestures agreed:** Fingers for 1-5; how to show 6-10?
- [ ] **Action gestures agreed:** Cut (scissors), Press (push), Hold (fist), Release (open hand), Rotate (circular motion)
- [ ] **Yes/No signals:** Thumbs up = yes/correct, Thumbs down = no/wrong
- [ ] **Repeat signal:** Circular hand motion = "show me that gesture again"
- [ ] **Urgency signal:** Rapid waving = "hurry / time is running out"

## Lobby Setup

- [ ] All three players are Steam friends with each other
- [ ] Host launches the game → "Host Game"
- [ ] Host invites both friends via Steam overlay (Shift+Tab → Friends → Invite to Game)
- [ ] Friends accept and appear in lobby
- [ ] Each player selects a **different** role (no duplicates)
- [ ] Everyone can see each other in the lobby
- [ ] Voice chat indicator shows active for Blind and Deaf

## First Bomb: What to Do

1. [ ] **Blind** does an initial sweep: reports module count and rough positions
2. [ ] **Deaf** identifies all module types by name before defusing starts
3. [ ] **Mute** locates all module solutions in the manual before the first gesture
4. [ ] **Start with the hardest module** — when time pressure is lowest
5. [ ] **Use the confirmation loop**: Instruction → Repeat → Confirm → Execute
6. [ ] **Deaf announces timer** at 60s, 30s, and 10s remaining

## After Each Bomb

- [ ] **Debrief:** What worked? What broke? Which gesture was confusing?
- [ ] **Celebrate small wins:** Even if you exploded, you learned something
- [ ] **Consider role rotation** after 3-4 bombs (optional but recommended)
- [ ] **Check team energy:** Tired teams make more mistakes. Take breaks.

> **Print this checklist** or keep it open on a second screen during your first session.
