## Communication in BOMBANANA!

Communication is the **real game** in BOMBANANA! The bomb modules are puzzles, but the challenge is getting the solution from the manual to the bomb through two layers of disabled communication.

## The Communication Chain

```
MANUAL  →  MUTE MONKEY  →  DEAF MONKEY  →  BLIND MONKEY  →  BOMB
(paper)    (gestures)      (speech)         (touch)
```

Each arrow is a potential failure point. Your goal is to minimize information loss at each step.

## Building a Gesture System

### Basic Gesture Categories

| Category | Examples |
|----------|----------|
| **Colors** | Point to clothing, use colored cards, pre-agreed hand shapes |
| **Numbers** | Finger counting (1-5 one hand, 6-10 two hands) |
| **Directions** | Pointing: up / down / left / right / center |
| **Actions** | Cut = scissors gesture, Press = push down, Hold = clenched fist |
| **Status** | Yes = thumbs up, No = thumbs down, Wait = open palm |
| **Urgency** | Speed of gesture, waving, pointing at timer |

### Advanced: Gesture Encoding

For complex modules, encode information efficiently:

- **Wire positions**: Number them 1-N from left to right
- **Colors**: Assign numbers to colors (1=red, 2=blue, 3=yellow, 4=green)
- **Module references**: Name each module position (TopLeft, MidRight, BottomCenter)

## Standard Phrase Templates

Train your team to use these standardized phrases:

| Situation | Deaf Monkey Says |
|-----------|-----------------|
| Starting a module | "Module [Name], [N] steps" |
| Giving an action | "[Action] the [position] [object]" |
| Confirming | "Confirm: you [action] the [object]?" |
| Wrong action | "Stop. Undo. New instruction:" |
| Module complete | "[Module name] done. Moving to [next module]" |

## Role-Specific Communication Tips

### Mute Monkey
- Position yourself where the Deaf can see you clearly
- Use **exaggerated movements** — small gestures are easily missed
- **Pause between gestures** to let the Deaf process
- Have a "reset/start over" gesture ready

### Deaf Monkey
- **Face the Mute** at all times, glance at bomb only when needed
- Speak in **short, imperative sentences**
- Always get **verbal confirmation** from Blind before proceeding
- Announce **timer status** regularly

### Blind Monkey
- **Vocalize everything** — "I'm touching the top wire now"
- Ask for **specific guidance** — "Which hand do I use?"
- Report **tactile feedback** — "This button feels stuck"
- **Count down** during timed actions

## Communication Drills

Practice these drills before real bombs:

1. **Color drill**: Mute gestures a color → Deaf names it → Blind points to it
2. **Number drill**: Mute gestures a number → Deaf says it → Blind holds up that many fingers
3. **Position drill**: Mute points to a position → Deaf describes it → Blind touches it
4. **Action chain**: Mute gestures cut + position → Deaf commands → Blind mimes it
