## Standard Team Callouts

Words matter — especially when a bomb is ticking. These standardized callouts eliminate ambiguity and speed up communication. Every team should use these exact phrases.

![Team Communication](/images/screenshots/ss_353f30e2c12b406ab6f48642e6c21b2c5ffe1d72.thumb.jpg)

## Why Standardized Callouts Matter

In high-pressure situations, people revert to habit. If your team has pre-agreed phrases, you don't waste mental energy constructing sentences — you just say the words you've practiced. This is the same principle used by pilots, surgeons, and military teams.

## Deaf Monkey Callouts

These are phrases the Deaf Monkey says out loud. The Blind hears them; the Mute hears them but responds only with gestures.

### Bomb Status Callouts

| Situation | Exact Phrase |
|-----------|-------------|
| Timer: 60 seconds remaining | "One minute — [N] modules remaining" |
| Timer: 30 seconds remaining | "Thirty seconds" |
| Timer: 10 seconds remaining | "TEN — execute, no confirm" |
| Strike occurred | "Strike [N] — [N] remaining" |
| Module complete | "[Module name] — green. Next: [next module]" |
| All modules complete | "ALL MODULES GREEN — bomb defused!" |

### Instruction Callouts (to Blind)

| Action | Exact Phrase Format |
|--------|-------------------|
| Cut wire | "Cut — [position] wire from [left/right]" |
| Press button | "Press — row [N], column [N]" |
| Hold button | "Hold — row [N], column [N] — I'll count down" |
| Release button | "Release — NOW" |
| Flip switch | "Flip switch [N] — [up/down]" |
| Rotate dial | "Rotate — [clockwise/counter-clockwise] — [N] clicks" |

### Confirmation Callouts

| Situation | Exact Phrase |
|-----------|-------------|
| Blind's action is correct | "Confirmed" |
| Blind is on wrong position | "Stop — move [left/right/up/down] — there" |
| Need Blind to repeat last action | "Again — same action" |
| Blind should wait | "Hold position — wait for next instruction" |

## Blind Monkey Callouts

These are what the Blind says — the Deaf hears them; the Mute hears them.

### Sweep Callouts

| Situation | Exact Phrase |
|-----------|-------------|
| Starting sweep | "Sweeping — hands on bomb" |
| Found a module | "Module found — [rough shape/size description]" |
| Sweep complete | "Sweep complete — [N] modules" |

### Execution Callouts

| Situation | Exact Phrase |
|-----------|-------------|
| Located target | "On [position] — confirm?" |
| Executing action | "[Action] now — [position]" |
| Action complete | "Done — next?" |
| Uncertain about position | "Not sure I'm on the right [wire/button] — verify" |

### Status Callouts

| Situation | Exact Phrase |
|-----------|-------------|
| Lost position | "Lost — resetting hands. Guide me back" |
| Made a mistake | "ERROR — hands off — awaiting instruction" |
| Something feels wrong | "Feels wrong — [description of tactile feedback]" |

## Mute Monkey Signals (Non-Verbal)

The Mute doesn't speak, but uses standardized body language:

| Signal | Meaning |
|--------|---------|
| Thumbs up | "Yes / Correct / Continue" |
| Thumbs down | "No / Wrong / Stop" |
| Circular hand motion | "Repeat that gesture" |
| Rapid waving | "Urgent — time is critical" |
| Pointing at own eyes, then bomb | "Look at the bomb" |
| Pointing at manual, then nodding | "I found the solution — watch me" |
| Both palms up, shrug | "I don't know / I can't find it in the manual" |
| Clap once | "Module complete — move to next" |

## Pre-Made Phrase Cards

Consider printing these callout cards and keeping them visible during play:

**Deaf Card:**
```
"Cut — [N]th wire from [left/right]"
"Press — row [N], column [N]"
"Confirmed"
"Stop — move [direction]"
"[N] seconds"
"Strike [N] — [N] remaining"
```

**Blind Card:**
```
"On [position] — confirm?"
"[Action] now"
"Done — next?"
"ERROR — hands off"
"Lost — resetting"
```

## Practice Drill

Run this 3-minute drill before each session:

1. **Deaf** reads a random instruction from the callout table
2. **Blind** repeats it back in the confirmation format
3. **Deaf** says "Confirmed" or corrects
4. Repeat 5 times, rotating which role reads
