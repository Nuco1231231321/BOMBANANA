## Multiplayer & Co-op

### Is BOMBANANA! multiplayer only?

Yes. BOMBANANA! is designed exclusively as a **3-player online co-op game**. There is no single-player mode and no 2-player mode. The asymmetric role system — blind, deaf, and mute monkeys — requires exactly three players, each filling one role.

### Does BOMBANANA! require exactly 3 players?

**Yes, exactly 3 players.** No more, no less. The game is balanced around the three-role system. You cannot play with 2 players (one role would be missing, breaking the communication chain) or 4 players (there are only 3 roles).

### Is there online co-op?

**Yes.** BOMBANANA! supports **Online Co-op** via Steam. All three players connect through Steam's multiplayer infrastructure. There is no need for port forwarding or third-party tools — just invite friends through the Steam overlay.

### Is there local / couch co-op?

Currently, BOMBANANA! does not support local split-screen or couch co-op. All three players need their own PC and their own copy of the game (or the free Demo). Each player needs their own screen because each role sees a completely different interface.

### Can you play solo?

No. The game cannot be played solo. Every role is essential — the Blind Monkey cannot see the manual, the Mute Monkey cannot touch the bomb, and the Deaf Monkey cannot hear responses. Playing solo would require controlling three different role interfaces simultaneously, which the game does not support.

### Is there crossplay?

BOMBANANA! is currently available only on **Steam (PC)**. Since there is only one platform, crossplay is not applicable at this time. Console versions have not been announced.

## Voice Chat & Communication

### Does BOMBANANA! have built-in voice chat?

**Yes.** The game has **in-game voice chat that is restricted by role**. This is a core mechanic:

- The **Blind Monkey** can hear and speak normally
- The **Deaf Monkey** can speak but **cannot hear** any voice chat
- The **Mute Monkey** can hear but **cannot speak** — their microphone is disabled

These restrictions are enforced by the game itself. Players must build a communication system using gestures (Mute → Deaf) and verbal instructions (Deaf → Blind).

### Can we use Discord instead of in-game voice chat?

Technically yes, but using external voice chat **defeats the core game mechanic**. The role-based voice restrictions are what make BOMBANANA! unique. If everyone can hear and speak freely on Discord, the Mute Monkey can just talk, and the game becomes a simple bomb-defusal puzzle with no communication challenge. Most players recommend using the in-game system for the intended experience.

### How does the Mute Monkey communicate?

The Mute Monkey uses **body language and gestures** to convey information. Before each session, teams should agree on a gesture vocabulary — what each hand signal means, how to indicate colors, numbers, directions, and actions. The Mute points, waves, holds up fingers, and mimes actions. The Deaf Monkey watches and interprets.

## Gameplay & Mechanics

### Is BOMBANANA! beginner-friendly?

BOMBANANA! has a learning curve, but the early campaign levels are designed to introduce mechanics gradually. The first few bombs have simple modules (3-4 per bomb, basic logic). The challenge comes from communication, not from complex puzzle mechanics. Most teams can successfully defuse their first bomb within 3-4 attempts.

### Can I create my own bombs?

**Yes!** Free Mode lets you design custom bombs from scratch. You choose the modules, set the timer, configure strike limits, add panic effects, and save your creation. You can share custom bombs with friends or the community via Steam.

### What languages does BOMBANANA! support?

The game supports: **English, Simplified Chinese, Japanese, Korean, Turkish, French, German, Spanish, Portuguese (Brazil), Russian, Polish, Italian, Dutch, Vietnamese, and Czech**.

### Does the game have accessibility features?

Yes. The Steam store page lists accessibility features including:
- **Camera Comfort** options
- **Playable without Timed Input** (some modules allow untimed choices)
- **Save Anytime** functionality
- **Keyboard Only** and **Mouse Only** options

### Is there controller support?

Currently, BOMBANANA! supports **keyboard and mouse**. Controller support has not been confirmed for the Demo but may be available in the full release.

### What are the system requirements?

**Minimum:**
- OS: Windows 10/11 64-bit
- Processor: Intel Core i3-6100 / AMD Ryzen 3 1200
- Memory: 2 GB RAM
- Graphics: Intel HD Graphics 520 / AMD Radeon Vega 3
- Storage: 300 MB

**Recommended:**
- Processor: Intel Core i5-8400 / AMD Ryzen 5 2600
- Memory: 8 GB RAM
- Graphics: Intel Iris Xe / NVIDIA GTX 1050 / AMD RX 560

The game is very lightweight and runs on most modern PCs.

### How many modules are in the game?

The Demo includes a selection of module types. Each bomb features 3-8 modules depending on difficulty. The full game (August 2026) will include significantly more module types and combinations.
